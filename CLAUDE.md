# Claude Code Instructions

## Package Manager

Ce projet utilise **pnpm**. Ne pas utiliser npm ou yarn.

## Commandes

```bash
pnpm build    # Build le projet
pnpm test     # Lance les tests (Jest)
pnpm lint     # Vérifie le linting
```

## Avant de considérer que tu as terminé:

- [ ] Vérifier que les tests passent
- [ ] Vérifier que le linting passe
- [ ] Vérifier que le build passe
